# 🌐 Generosity Network

## 📖 About the Project
Generosity Network is a web platform designed to connect charitable organizations with people who want to make a difference. The idea behind this project is simple — make it easier for communities to discover events, volunteer, and support causes in a transparent and organized way.

Instead of scattered communication and manual coordination, this platform brings everything into one place.

---

## 🚀 What You Can Do

### 👥 For Users
- Explore upcoming charity events
- Register as a volunteer
- Participate in events
- Donate to specific causes
- Share feedback or suggest new events

### 🛠️ For Admin
- Create and manage events
- Add event details (date, time, location)
- Include donation/payment information
- View registered volunteers
- Read feedback and suggestions from the community

---

## ✨ Key Features
- Community-driven platform  
- Easy volunteer registration  
- Event creation and tracking  
- Donation support for events  
- Admin dashboard for full control

## ⚙️ How to Run the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/Keerthana-Chowdary/Generosity-Network.git
2. Move the project folder to your local server:
        Copy the folder into xampp/htdocs
3. Import the database:
        Open phpMyAdmin
        Create a new database (e.g., generosity_network)
        Import the provided .sql file into the database
4. Start the server:
        Open XAMPP
        Start Apache and MySQL
5. Run the project:
        Open your browser and go to:
        http://localhost/Generosity-Network
